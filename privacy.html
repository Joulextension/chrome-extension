What this site collects

In the current static release, this site does not submit real personal data. In the planned integrated version, it may process account identifiers such as email address, billing status, plan metadata and support messages.

Authentication and billing providers

Future production integrations are expected to use Firebase Auth for sign-in, a Cloudflare Worker API for account access, D1 for account records and Stripe for payment processing and invoice management.
